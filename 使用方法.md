# Latex数学公式
总体上说就是两个$扩起来
1. **数学符号：**
   - `^` 表示上标（例如，`x^2` 表示 x 的平方）。 
   - `_` 表示下标（例如，`a_1` 表示 a 的下标为1）。
   - `\frac{a}{b}` 表示分数，其中 a 是分子，b 是分母。
   - `\sqrt{n}` 表示平方根。
   - `\sum` 表示求和。
   - `\int` 表示积分。
   - `\infty` 表示无穷大。
$
    \frac{x^2+y_1}{\sqrt{x}}
$
1. **希腊字母：**
   - `\alpha`, `\beta`, `\gamma`, ... 表示希腊字母。

2. **关系符号：**
   - `=` 表示等于。
   - `\neq` 表示不等于。
   - `<`, `>`, `\leq`, `\geq` 分别表示小于、大于、小于等于、大于等于。

3. **集合和逻辑运算：**
   - `\cup`, `\cap` 表示并集和交集。
   - `\subset`, `\subseteq`, `\supset`, `\supseteq` 表示子集和超集。
   - `\in`, `\notin` 表示属于和不属于。

4. **矩阵：**
   - `\begin{matrix} ... \end{matrix}` 用于创建矩阵。

5. **分隔符：**
   - `{}`, `()` 表示大括号和圆括号。
   - `[]` 表示方括号。
   - `\langle`, `\rangle` 表示尖括号。
